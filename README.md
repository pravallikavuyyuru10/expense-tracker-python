# Expense Tracker Python

## Overview

Expense Tracker is a simple Python application that helps users record and analyze their expenses.

The program allows users to enter multiple expenses and automatically calculates important financial insights such as total spending, highest expense, lowest expense, and average expense.

---

## Features

* Add multiple expenses
* Calculate total expenses
* Find the highest expense
* Find the lowest expense
* Calculate average expense
* Simple command-line interface

---

## Technologies Used

* Python

---

## Concepts Used

* Variables
* User Input
* Lists
* Loops
* Conditional Statements
* Functions
* Basic Data Analysis

---

## How It Works

1. User enters the number of expenses.
2. User enters each expense amount.
3. The program stores expenses in a list.
4. The program calculates:

   * Total Expense
   * Highest Expense
   * Lowest Expense
   * Average Expense
5. Results are displayed to the user.
